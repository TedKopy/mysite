Things and Links

Here I will paste some links and references I consider useful. Most of them will also probably deserve some feedback. I decided to put them all in a table.

Resource Name Actual link Description
Sololearn https://www.sololearn.com/ It is one of the first resources that helped me to dig into the programming. It is not enough, of course, however, it will surely give you the basics. Oh, and they've got some apps for your smartphone as well.
Learn Enough https://www.learnenough.com/ Some pretty easy manuals that included Linux CLI, Text Editors, GitHub itself and HTML. Be aware that some of them may be not ready yet. Anyway, they are cool at least because they make you believe you can do it.
W3Schools https://www.w3schools.com/ You can't keep all the things about HTML and CSS in your head. Even if so, it can be quite difficult. So sometimes it is better to google it. And prior to googling it, you can always try lurking this extremely helpful website
LinuxCommand http://linuxcommand.org/ My first introduction to Linux can be tightly related to this website, and though I gave up pretty quickly, I still think I should come back later and learn some more.
